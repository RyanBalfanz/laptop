# RyanBalfanz/laptop

A shell script to set up a macOS laptop for web and mobile development.

[![Build Status](https://travis-ci.org/RyanBalfanz/laptop.svg?branch=master)](https://travis-ci.org/RyanBalfanz/laptop)
